# 📊 Hospitality Revenue Dashboard

## 📌 Overview
This project focuses on analyzing hotel revenue performance using Power BI.  
The dashboard provides meaningful insights into business performance by tracking key metrics such as revenue, occupancy, and pricing trends.

---

## 🎯 Objective
- Analyze hotel booking and revenue data  
- Monitor key performance indicators (KPIs)  
- Provide data-driven insights to improve business decisions  

---

## 📊 Key Metrics (KPIs)
- **RevPAR (Revenue per Available Room)**  
- **ADR (Average Daily Rate)**  
- **Occupancy Rate**  

---

## 🛠️ Tools & Technologies
- Power BI  
- DAX (Data Analysis Expressions)  
- Power Query  

---

## 🚀 Features
- Interactive dashboard with dynamic filtering  
- KPI cards for quick performance tracking  
- Trend analysis using charts and visualizations  
- Filters based on city, hotel, and time period  

---

## 📈 Key Insights
- Identified peak seasons with higher occupancy rates  
- Observed pricing patterns impacting revenue  
- Suggested strategies that could improve revenue by ~15%  

---

## 📂 Csv files
- The csv files contains hotel booking and revenue details  
- Includes attributes like date, city, hotel type, occupancy, and pricing
